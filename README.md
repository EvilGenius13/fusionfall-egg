## Usage

1. **Download the Egg**: Get `egg-pterodactyl-open-fusion.json` from this repository.
2. **Import to your game panel**:
   - Go to your game panel (Pterodactyl or Pelican).
   - Import the egg.
   - Select the JSON file you downloaded.
3. **Create Server**: Deploy a new server using the OpenFusion egg.

## Extra Information
There are a few extra variables that can be set in the panel to configure the server.

- **ANNOUNCE_IP**: The IP sent to the client. Set this to your LAN IP (192.168.x.x). If your host is bound to 0.0.0.0 you probably need this.
- **LOGIN_PORT**: The port for the login server.
- **SHARD_PORT**: The port for the shard server.
- **MONITOR_PORT**: The port for the monitor server.
